Задача
- Пульсирование логотипа сделать управляемым
- Начинать анимацию только после загрузки
- Подгружать остальной html только после загрузки страницы и главное изображений
- Плавное появление страницы загрузки

Что за последовательность будет при загрузке?
- В html странице будет только экран загрузки.
- Дальше призагрузке страницы этот экран будет открыт, а затем показан.
- После, подгрузится остальной html и экран загрузки по окончанию подгрузки, будет закрыт.

Метод show
Этот метод должен проиграть анимацию свойства opacity элемента экрана загрузки. Принцип работаы примерно такой. Разворачивается функция интевала. Что она делает? Эта функция устанавливает свойство opacity и прибавляет к нему еще значение. При последующих итерациях, при условии, что значение opacity достигло полноты, интервал останавливается.

Нужен контроллер загрузки
Достаточно просто метода инициализации выполненного в виде Promise. Тогда он получит от сервера html разметку и вставит ее на свое место. Кстати нужна еще и модель загрузки. Это для того чтобы получить сам текст разметки.
